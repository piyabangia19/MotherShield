# 🛡️ MotherShield

An AI-powered maternal risk assessment system designed to assist in early detection, monitoring, and intervention for maternal health risks.

---

## 🚀 Overview

MotherShield is a smart healthcare solution that uses AI agents to analyze maternal health data and provide insights. It helps in identifying risks during pregnancy and postpartum.

---

## ✨ Features

* AI-based Risk Detection
* Dashboard Visualization
* Alert & Referral System
* Facility Recommendation
* PDF Report Generation

---

## 🧠 Tech Stack

* React + Vite
* Tailwind CSS
* JavaScript

---

## ⚙️ Run Locally

```bash
git clone https://github.com/piyabangia19/MotherShield.git
cd MotherShield
npm install
npm run dev
```

---

## 🔒 Environment Variables

Create `.env` file (don’t upload it):

```
VITE_API_KEY=your_key
```
